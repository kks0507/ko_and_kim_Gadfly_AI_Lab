# /gadfly-new-project

새로운 연구 프로젝트를 생성합니다. 대화형으로 핵심 정보를 입력받아 프로젝트 설정 파일을 완성합니다. 모든 학문 분야를 지원합니다.

## 사용법

```
/gadfly-new-project <프로젝트명>
```

## 실행 절차

### Phase 1: 템플릿 복사

`projects/_template.md`를 `projects/<프로젝트명>.md`로 복사합니다.

### Phase 2: 대화형 정보 입력

AskUserQuestion 도구를 사용하여 효율적으로 정보를 수집합니다:

1. **연구 도메인**: 자유 입력 (교육공학, 문헌정보학, 심리학, 경영학, 의학, 핀테크 등)
2. **교육 대상** (해당 시): 초중등 / 고등교육 / 성인교육 / 해당없음
3. **연구 개요**: 이 연구가 무엇을 하려는지 2-3문장
4. **핵심 키워드**: 영어/한국어 키워드
5. **이론적 프레임워크**: 기반 이론/모델
6. **독창적 주장**: 기존 연구와 차별화 포인트
7. **비판 프레임**: 기존 연구를 비판할 렌즈
8. **방법론 선호도**: 양적/질적/혼합/미정
9. **현재 연구 단계**: 아이디어/선행연구조사/방법론설계/데이터수집/분석/집필/투고

### Phase 3: 프로젝트 파일 생성

수집된 정보로 프로젝트 설정 파일을 완성하고 저장합니다.

### Phase 4: output 디렉토리 생성

`output/<프로젝트명>/` 디렉토리를 생성합니다.

### Phase 5: 확인

```
✓ 프로젝트 생성 완료: <프로젝트명>
  - 설정 파일: projects/<프로젝트명>.md
  - 출력 폴더: output/<프로젝트명>/

이제 다음 커맨드로 연구를 시작할 수 있습니다:
  /gadfly-brainstorm <아이디어>                   아이디어 구체화
  /gadfly-search-strategy <주제>                  검색 전략 수립
  /gadfly-cite <논문.pdf> --project <프로젝트명>   논문 정밀 인용
  /gadfly-analyze <논문.pdf> --project <프로젝트명> 논문 종합 분석
```
