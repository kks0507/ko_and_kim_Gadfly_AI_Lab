# /gadfly-new-project

새로운 연구 프로젝트를 생성합니다. 대화형으로 핵심 정보를 입력받아 프로젝트 설정 파일을 완성합니다.

## 사용법

```
/gadfly-new-project <프로젝트명>
```

## 실행 절차

### Phase 1: 템플릿 복사

`projects/_template.md`를 `projects/<프로젝트명>.md`로 복사합니다.

### Phase 2: 대화형 정보 입력

사용자에게 순차적으로 질문하여 프로젝트 설정을 채웁니다:

1. **연구 도메인**: 교육공학+AI / 문헌정보학+AI / 핀테크+AI / 기타
2. **교육 대상** (해당 시): 초중등 / 고등교육 / 성인교육
3. **연구 개요**: 이 연구가 무엇을 하려는지 2-3문장
4. **핵심 키워드**: 영어/한국어 키워드
5. **이론적 프레임워크**: 기반 이론/모델
6. **독창적 주장**: 기존 연구와 차별화 포인트
7. **비판 프레임**: 기존 연구를 비판할 렌즈

### Phase 3: 프로젝트 파일 생성

수집된 정보로 프로젝트 설정 파일을 완성하고 저장합니다.

### Phase 4: output 디렉토리 생성

`output/<프로젝트명>/` 디렉토리를 생성합니다.

### Phase 5: 확인

```
✓ 프로젝트 생성 완료: <프로젝트명>
  - 설정 파일: projects/<프로젝트명>.md
  - 출력 폴더: output/<프로젝트명>/

이제 다음 커맨드로 논문 분석을 시작할 수 있습니다:
  /gadfly-analyze <논문> --project <프로젝트명>
```
