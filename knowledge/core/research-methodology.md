# 연구방법론 개요 (Research Methodology)

## 핵심 개념

### 양적 연구 (Quantitative Research)
- 가설 검증 중심의 연역적(Deductive) 접근
- **실험 연구(Experimental)**: 무작위 배정(Random Assignment), 통제 집단, 인과 관계 추론 가능
- **준실험 연구(Quasi-experimental)**: 무작위 배정 불가 시, 비동등 통제 집단 설계
- **조사 연구(Survey)**: 대규모 표본, 횡단(Cross-sectional)/종단(Longitudinal) 설계
- **상관 연구(Correlational)**: 변인 간 관계 탐색, 인과 추론 불가

### 질적 연구 (Qualitative Research)
- 현상의 의미와 맥락을 탐구하는 귀납적(Inductive) 접근
- **사례 연구(Case Study)**: 특정 사례의 심층적 분석, 경계 설정(Bounding) 필수 (Yin, 2018)
- **문화기술지(Ethnography)**: 장기 현장 참여, 문화적 맥락의 두터운 기술(Thick Description)
- **현상학(Phenomenology)**: 체험의 본질 구조 탐구, 판단 중지(Epoché)
- **근거이론(Grounded Theory)**: 개방/축/선택 코딩, 이론 생성 목적 (Strauss & Corbin)
- **내러티브 탐구(Narrative Inquiry)**: 경험의 이야기적 재구성

### 혼합 연구 (Mixed Methods)
- 양적 + 질적 방법의 통합적 활용 (Creswell & Plano Clark)
- **수렴적 설계(Convergent)**: 양적/질적 동시 수집 후 결과 병합
- **설명적 순차 설계(Explanatory Sequential)**: 양적 → 질적 (양적 결과 심층 해석)
- **탐색적 순차 설계(Exploratory Sequential)**: 질적 → 양적 (탐색 후 검증)
- **변환적 설계(Transformative)**: 사회 정의 프레임워크 기반

### 디자인 기반 연구 (Design-Based Research, DBR)
- 교육 현장의 반복적 설계-실행-분석 순환
- 이론과 실천의 동시적 발전 추구
- 맥락 의존적(Context-dependent), 생태학적 타당도(Ecological Validity) 강조
- 설계 원리(Design Principles) 도출이 핵심 산출물

## 주요 프레임워크

### Creswell의 연구 설계 프레임워크
- **세계관(Worldview)**: 후기실증주의, 구성주의, 변환적, 실용주의
- **연구 접근(Approach)**: 양적, 질적, 혼합
- **연구 설계(Design)**: 구체적 전략 및 절차
- 이론 → 세계관 → 접근 → 설계 → 방법의 정합성(Alignment) 검증 필수

### 이론-방법론 정합성 검증 체크리스트
| 검증 항목 | 질문 |
|-----------|------|
| 존재론적 정합성 | 연구하려는 현실의 본질에 이 방법론이 부합하는가? |
| 인식론적 정합성 | 지식 생성 방식이 이론적 전제와 일치하는가? |
| RQ-방법론 정합성 | 연구 질문의 유형(what/how/why)에 방법론이 적합한가? |
| 데이터 적합성 | 수집하려는 데이터가 이론적 구성개념을 포착할 수 있는가? |
| 분석 적합성 | 분석 방법이 이론적 가정과 일관되는가? |

## 방법론별 비교

| 방법론 | 적합한 RQ 유형 | 데이터 수집 | 분석 방법 | 장점 | 한계 |
|--------|---------------|-------------|-----------|------|------|
| 실험 연구 | 인과 관계(X→Y) | 검사, 설문 | 통계 분석 | 내적 타당도 높음 | 생태학적 타당도 낮음 |
| 사례 연구 | How/Why 탐색 | 면담, 관찰, 문서 | 주제 분석 | 맥락 풍부 | 일반화 제한 |
| 근거이론 | 이론 부재 영역 | 면담, 관찰 | 코딩(개방/축/선택) | 이론 생성 | 시간 소요 큼 |
| 혼합 연구 | 복합적 현상 | 복합 데이터 | 통합 분석 | 상호보완 | 설계 복잡성 |
| DBR | 설계 원리 도출 | 반복 순환 데이터 | 반복적 분석 | 실천적 | 재현 어려움 |

## 비판적 관점
- 양적/질적 이분법의 한계: 실용주의(Pragmatism) 관점에서의 통합 필요
- 서구 중심 연구 패러다임의 비서구 맥락 적용 시 문화적 재해석 필요
- WEIRD(Western, Educated, Industrialized, Rich, Democratic) 표본 편향 문제
- 한국 교육 연구 맥락에서의 혼합 연구 증가 추세와 방법론적 엄밀성 논의
- 재현 가능성(Reproducibility) 위기와 사전등록(Pre-registration) 운동

## 용어 사전
| 영어 | 한국어 | 설명 |
|------|--------|------|
| Random Assignment | 무작위 배정 | 참여자를 실험/통제 집단에 무작위 할당 |
| Thick Description | 두터운 기술 | 맥락과 의미를 풍부하게 포함한 질적 기술 |
| Triangulation | 삼각검증 | 다양한 자료원/방법을 통한 타당성 확보 |
| Ecological Validity | 생태학적 타당도 | 연구 결과의 실제 상황 적용 가능성 |
| Member Checking | 참여자 확인 | 질적 연구에서 참여자에 의한 결과 검증 |
| Saturation | 포화 | 질적 자료에서 새로운 정보가 더 이상 나타나지 않는 상태 |
| Pre-registration | 사전등록 | 연구 수행 전 가설/분석 계획을 공개 등록 |
| Epoché | 판단 중지 | 현상학에서 선입견을 배제하는 태도 |
