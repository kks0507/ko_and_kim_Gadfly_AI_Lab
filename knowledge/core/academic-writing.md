# 학술 글쓰기 (Academic Writing)

## 핵심 개념

### APA 7th 형식 기본 원칙
- **원고 구성**: 제목 페이지 → 초록(Abstract) → 본문 → 참고문헌 → 부록
- **본문 구조(IMRaD)**: 서론(Introduction) → 방법(Method) → 결과(Results) → 고찰(Discussion)
- **제목 수준**: Level 1(볼드, 중앙정렬) → Level 2(볼드, 왼쪽정렬) → Level 3(볼드 이탤릭, 왼쪽정렬)
- **글꼴/여백**: 12pt, 이중 간격(Double-spaced), 1인치 여백
- **편향 없는 언어(Bias-free Language)**: 인종, 성별, 장애 관련 표현 가이드라인 준수

### 결과 섹션 구조 (Results Section)
1. **예비 분석(Preliminary Analyses)**
   - 데이터 스크리닝: 결측치 처리, 이상치 확인
   - 가정 검증: 정규성, 등분산성 검토
2. **기술통계(Descriptive Statistics)**
   - 주요 변인의 평균, 표준편차, 상관행렬
   - 표(Table)로 제시 권장
3. **추론통계(Inferential Statistics)**
   - 가설별 검정 결과, 통계량, 유의확률
   - APA 형식 준수: *t*, *F*, *p* 등 이탤릭 표기
4. **효과크기(Effect Size) 및 신뢰구간(CI)**
   - 모든 주요 검정에 효과크기 보고 필수
   - 95% CI 병기 권장

### 고찰 섹션 구조 (Discussion Section)
1. **주요 발견 요약**: 가설 지지/기각 여부, 핵심 결과 간략 정리 (통계량 반복 지양)
2. **선행연구와 비교**: 일치/불일치 결과에 대한 해석, 이유 탐색
3. **이론적 함의(Theoretical Implications)**: 기존 이론의 확장, 수정, 정교화
4. **실천적 함의(Practical Implications)**: 교육 현장, 정책, 실무에 대한 시사점
5. **연구의 한계(Limitations)**: 표본, 방법론, 측정 도구 관련 제한점 솔직히 기술
6. **후속 연구 제안(Future Research)**: 한계 극복 방안 + 새로운 연구 방향 제시

### 표와 그림 설계 (APA Tables and Figures)
- **표(Table)**: 번호와 제목(이탤릭) 표 상단, 주석(Note) 표 하단
  - 세로선 금지, 가로선은 최소화 (상단선, 열 구분선, 하단선만)
  - 소수점 자릿수 통일, 단위 명시
- **그림(Figure)**: 번호와 제목(이탤릭) 그림 상단 (APA 7th 변경사항)
  - 축 레이블 명확, 범례 포함, 흑백 인쇄 고려
  - 오차 막대(Error bars) 포함 권장

## 주요 프레임워크

### 인용 통합 기법 (Citation Integration)
| 유형 | 설명 | 예시 |
|------|------|------|
| 내러티브 인용(Narrative) | 저자명이 문장의 주어/목적어 | Kim(2023)은 ~을 보고하였다 |
| 괄호 인용(Parenthetical) | 문장 끝 괄호 안에 출처 | ~으로 나타났다(Kim, 2023) |
| 패러프레이즈(Paraphrase) | 원문을 자신의 언어로 재구성 | 선행 연구에서는 ~을 강조하였다(Kim, 2023) |
| 직접 인용(Direct Quote) | 원문 그대로 인용 (페이지 번호 필수) | "원문 그대로"(Kim, 2023, p. 45) |

- **인용 비율**: 패러프레이즈 중심(80% 이상), 직접 인용 최소화
- **인용 동사 다양화**: 주장하였다, 보고하였다, 제안하였다, 발견하였다, 강조하였다

### 기여 작성법 (Contribution Statement)
| 기여 유형 | 작성 포인트 |
|-----------|------------|
| 이론적 기여(Theoretical) | 기존 이론의 확장/수정/통합, 새로운 개념/프레임워크 제안 |
| 방법론적 기여(Methodological) | 새로운 측정 도구, 분석 방법, 연구 설계의 혁신 |
| 실천적 기여(Practical) | 교육 현장 적용 가능한 구체적 설계 원리, 가이드라인 |

### 한국 학술지 특수 형식 요소
- **국문 초록 + 영문 초록**: 대부분의 KCI 학술지에서 양쪽 모두 요구
- **키워드**: 국문/영문 각 3-5개
- **참고문헌**: 국문 문헌 → 영문 문헌 순서 (학술지별 상이)
- **저자 정보**: 소속, 직위, 이메일, 교신저자(Corresponding Author) 표시
- **분량 제한**: 학술지별 상이 (보통 20-25쪽, A4 기준)
- **학문 분야 분류**: 투고 시 KCI 학문 분류 코드 선택

## 비판적 관점
- APA 형식의 영어 중심성: 한국어 학술 글쓰기 관행과의 충돌
- 결과 섹션의 "유의미한 결과" 중심 보고 편향(Reporting Bias)
- 한계 섹션의 형식적 나열 vs 진정한 성찰적 기술의 차이
- 고찰에서의 과잉 일반화(Overgeneralization) 경계
- 학제간 연구(Interdisciplinary)에서 글쓰기 관행 충돌 문제

## 용어 사전
| 영어 | 한국어 | 설명 |
|------|--------|------|
| IMRaD | 서론-방법-결과-고찰 | 학술 논문의 표준 구조 |
| Paraphrase | 패러프레이즈 | 원문의 의미를 자신의 표현으로 재구성 |
| Narrative Citation | 내러티브 인용 | 저자명을 문장 구성 요소로 사용하는 인용 |
| Bias-free Language | 편향 없는 언어 | 차별적 표현을 배제한 학술 언어 사용 |
| Corresponding Author | 교신저자 | 논문 관련 연락을 담당하는 저자 |
| Theoretical Implications | 이론적 함의 | 연구 결과가 이론에 시사하는 점 |
| Effect Size | 효과크기 | 처치 효과의 실질적 크기를 나타내는 지표 |
| Reporting Bias | 보고 편향 | 유의한 결과만 선택적으로 보고하는 경향 |
