# 설문 설계 (Survey Design)

## 핵심 개념

### 문항 작성 원칙 (Item Writing Principles)
- **이중부정 금지(Avoid Double Negatives)**: "~하지 않는 것에 동의하지 않는다" 형태 배제
- **유도 질문 금지(Avoid Leading Questions)**: 특정 응답을 암시하는 표현 배제
- **단일 차원(Single Dimension)**: 하나의 문항에 하나의 개념만 측정 (이중 질문 Double-barreled 금지)
- **명확한 용어 사용**: 모호한 표현("자주", "가끔") 대신 구체적 빈도 제시
- **응답자 수준 고려**: 전문 용어 최소화, 문항 길이 적정화 (20단어 이내 권장)
- **사회적 바람직성 편향(Social Desirability Bias)** 통제: 간접 질문, 익명성 강조

### 척도 유형 (Scale Types)
- **리커트 척도(Likert Scale)**
  - 5점: 전혀 그렇지 않다(1) ~ 매우 그렇다(5) — 한국 연구에서 가장 보편적
  - 7점: 세밀한 변별력, 정규 분포 유도에 유리
  - 중립점 포함/배제 논쟁: 강제 선택 vs 중립 허용
- **의미분별 척도(Semantic Differential Scale)**: 양극 형용사 쌍 (예: 쉬운 ◻◻◻◻◻ 어려운)
- **시각 아날로그 척도(Visual Analog Scale, VAS)**: 연속적 응답 (통증, 만족도)
- **빈도 척도**: 전혀 없다 / 월 1-2회 / 주 1-2회 / 거의 매일

### 타당도 (Validity)
| 타당도 유형 | 검증 방법 | 핵심 기준 |
|------------|-----------|-----------|
| 내용타당도(Content) | 전문가 패널 검토 (3-5인), CVI(Content Validity Index) ≥ .78 | 문항이 구성개념을 충분히 대표하는가 |
| 구성타당도(Construct) | 탐색적 요인분석(EFA), 확인적 요인분석(CFA) | 이론적 요인 구조와 실제 자료의 부합 |
| 수렴타당도(Convergent) | AVE ≥ .50, 요인적재량 ≥ .50 | 같은 구성개념 측정 문항 간 높은 상관 |
| 변별타당도(Discriminant) | AVE > 요인 간 상관²  (Fornell-Larcker 기준) | 다른 구성개념 측정 문항과 구별 |
| 준거타당도(Criterion) | 외적 준거와의 상관 | 예측적/동시적 타당도 |

### 신뢰도 (Reliability)
- **내적 일관성(Internal Consistency)**: Cronbach's α ≥ .70 (Nunnally 기준), 최적 .80-.90
- **검사-재검사(Test-retest)**: 2-4주 간격, 상관 ≥ .70
- **반분 신뢰도(Split-half)**: Spearman-Brown 공식 활용
- **복합 신뢰도(Composite Reliability, CR)**: SEM 기반, CR ≥ .70

## 주요 프레임워크

### 설문 개발 절차
1. **구성개념 정의**: 이론적 배경에서 측정 변인 정의
2. **문항 풀 생성**: 선행연구 문항 검토 + 신규 문항 개발
3. **전문가 검토**: 내용타당도 검증 (CVI 산출)
4. **파일럿 테스트(Pilot Test)**: 소규모 표본(30-50명), 문항 분석
   - 문항-총점 상관(Item-total correlation) ≥ .30
   - 문항 삭제 시 α 변화 확인
   - 천장/바닥 효과(Ceiling/Floor effect) 검토
5. **탐색적 요인분석(EFA)**: 요인 구조 탐색 (표본 100-300명)
6. **확인적 요인분석(CFA)**: 요인 구조 확인 (새 표본 200명 이상)
7. **최종 도구 확정**: 타당도/신뢰도 증거 종합 보고

### IRB 체크리스트 (기관생명윤리위원회)
| 항목 | 확인 내용 |
|------|-----------|
| 연구 목적 명시 | 참여자에게 연구 목적과 절차를 명확히 안내 |
| 자발적 참여 | 참여 거부 및 중도 철회 시 불이익 없음을 명시 |
| 비밀보장 | 개인정보 보호, 데이터 익명화/부호화 처리 방법 기술 |
| 철회 권리 | 언제든 참여를 중단할 수 있으며 수집 자료 삭제 가능 |
| 위험/이익 | 참여에 따른 잠재적 위험과 이익 고지 |
| 보상 | 참여 보상이 있을 경우 내용과 조건 명시 |
| 자료 보관 | 수집 자료의 보관 기간, 보관 방법, 폐기 절차 명시 |
| 동의서 | 서면 또는 온라인 동의서, 연구자 연락처 포함 |

## 설문 배포 전략
- **온라인**: Google Forms, Qualtrics, SurveyMonkey, 네이버 폼
- **오프라인**: 인쇄 설문지, 현장 배포/회수
- **응답률 향상**: 사전 안내, 리마인더(1-2회), 적정 소요 시간(10-15분), 인센티브
- **표본 크기**: EFA — 문항 수의 5-10배, CFA — 최소 200명, SEM — 최소 200-500명

## 비판적 관점
- 자기보고식 설문의 고유 한계: 사회적 바람직성, 공통방법편의(Common Method Bias)
- 서구 개발 척도의 한국 맥락 적용 시 문화적 타당성(Cross-cultural Validity) 검증 필요
- 한국 응답자의 중앙 경향 편향(Central Tendency Bias): 극단 응답 회피 경향
- 온라인 설문의 성실 응답(Careless Responding) 판별 문제: 주의 확인 문항, 응답 시간 분석
- 번역-역번역(Translation-Back Translation) 절차의 형식화 비판

## 용어 사전
| 영어 | 한국어 | 설명 |
|------|--------|------|
| Likert Scale | 리커트 척도 | 동의 정도를 숫자로 표현하는 응답 형식 |
| Content Validity Index | 내용타당도 지수 | 전문가 평정에 기반한 문항 적합성 지수 |
| Cronbach's Alpha | 크론바흐 알파 | 문항 간 내적 일관성을 나타내는 신뢰도 계수 |
| Pilot Test | 파일럿 테스트 | 본조사 전 소규모 예비 시행 |
| Common Method Bias | 공통방법편의 | 동일 방법 사용으로 인한 인위적 상관 증가 |
| IRB | 기관생명윤리위원회 | 연구 참여자 보호를 위한 윤리 심의 기구 |
| Social Desirability | 사회적 바람직성 | 바람직하게 보이려는 응답 경향 |
| Floor/Ceiling Effect | 바닥/천장 효과 | 응답이 척도 양극단에 몰리는 현상 |
