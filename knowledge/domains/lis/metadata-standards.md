# 메타데이터 표준 (Metadata Standards)

## 핵심 개념

### 주요 표준
- **Dublin Core (DC)**: 15개 핵심 요소의 범용 메타데이터 표준
- **MARC 21**: 도서관 목록 레코드의 국제 표준
- **IEEE LOM**: 학습 객체 메타데이터 표준
- **Schema.org**: 웹 기반 구조화 데이터 표준
- **MODS/METS**: 디지털 객체 기술 및 구조 표준

### 지식 조직 체계
- 분류법(Classification): DDC, LCC, KDC
- 시소러스(Thesaurus): 통제어휘 기반 의미 관계
- 온톨로지(Ontology): 형식적 개념 정의 및 관계 명세
- 링크드 데이터(Linked Data): 웹 기반 지식 연결

### 상호운용성
- 메타데이터 매핑 및 크로스워크
- OAI-PMH (메타데이터 수확 프로토콜)
- FAIR 원칙 (Findable, Accessible, Interoperable, Reusable)

## 비판적 관점
- 서구 중심 분류 체계의 문화적 편향
- 다국어 메타데이터 처리 문제
- AI 생성 메타데이터의 품질 관리
- 전통적 표준과 현대 AI 시스템 간 갭

## 용어 사전
| 영어 | 한국어 | 설명 |
|------|--------|------|
| Dublin Core | 더블린 코어 | 15개 핵심 요소의 범용 메타데이터 표준 |
| Controlled Vocabulary | 통제어휘 | 표준화된 용어 집합 |
| Interoperability | 상호운용성 | 시스템 간 데이터 교환 가능성 |
| Linked Data | 링크드 데이터 | 웹에서 구조화된 데이터를 연결하는 방법 |
