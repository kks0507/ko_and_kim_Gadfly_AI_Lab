# 정보검색 (Information Retrieval)

## 핵심 개념

### 정보검색 모델
- **Boolean Model**: 불리언 연산 기반 정확 매칭
- **Vector Space Model**: TF-IDF 기반 유사도 계산
- **Probabilistic Model**: BM25 등 확률 기반 랭킹
- **Neural IR**: BERT, Dense Retrieval 등 딥러닝 기반 검색

### 학술 정보 검색
- 학술 데이터베이스 (Scopus, Web of Science, ERIC, LISTA, RISS)
- 인용 네트워크 분석
- 체계적 문헌 고찰(Systematic Review) 방법론
- PRISMA 가이드라인

### 학술정보관 서비스
- 디스커버리 시스템(Discovery System)
- 기관 리포지토리(Institutional Repository)
- 연구 데이터 관리(Research Data Management)
- 오픈 액세스(Open Access) 정책

## 주요 이론/프레임워크
- Information Foraging Theory
- Sense-Making (Dervin)
- Ellis's Model of Information Seeking
- Kuhlthau's Information Search Process

## 비판적 관점
- 알고리즘 편향에 의한 정보 접근 불평등
- 영어 중심 학술 생태계의 비영어권 배제
- 상업 데이터베이스 의존성
- AI 기반 검색의 투명성/설명 가능성

## 용어 사전
| 영어 | 한국어 | 설명 |
|------|--------|------|
| Information Retrieval | 정보검색 | 대규모 정보에서 관련 정보를 찾는 과정 |
| Discovery System | 디스커버리 시스템 | 통합 검색 인터페이스 |
| Precision/Recall | 정확률/재현율 | 검색 성능의 핵심 지표 |
| Systematic Review | 체계적 문헌 고찰 | 체계적 방법론에 기반한 문헌 분석 |
